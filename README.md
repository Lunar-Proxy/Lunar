## Terminal
> [!NOTE]
> You will need to install [git](https://git-scm.com/downloads), [nodejs](https://nodejs.org/en/download/prebuilt-installer) and pnpm by running `npm i -g pnpm`


```
git clone https://github.com/Lunar-Proxy/Lunar.git
cd Lunar
pnpm install
pnpm run build
pnpm start
```
